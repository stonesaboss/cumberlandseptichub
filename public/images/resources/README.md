# Resources Image Manifest

Image slots for the septic knowledge catalog. Components render a styled
placeholder until each file below is added AND its path is set in
`src/data/resources.ts` (`heroImage` for hubs, `RESOURCES_HOME.heroImage`
for the landing page). Once the path is set, the image displays automatically.

**Format:** WebP, landscape 16:9, recommended source dimensions 1600×900
(cards render smaller; one source size covers all uses).
Keep files under ~300 KB where possible.
